# SecureOps Website Assets

Original generated raster assets for the SecureOps phase 1 cybersecurity website.

Use `assets-manifest.json` for recommended routes, alt text, and placement.

Suggested app placement after the Next.js project is scaffolded:

```text
public/assets/secureops/
```

Notes:

- Images are WebP and contain no readable text.
- Add headings, CTAs, and SEO copy in HTML, not inside the images.
- `asset-contact-sheet.webp` is only for preview and should not be used in production UI.
- These assets are Sophos-inspired in quality and category, but do not use Sophos logos, copy, or proprietary images.
